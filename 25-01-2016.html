Java - Sychronized arguments

  1. In specified class, create "private static <var_type> <var_name>"
  2. In specified class, create "public static synchronized <return_type> <function_name>"
    3. Now the synchronized functions can be called in multiple thread objects. DONE.

Java - Generate random number 
double start = 400;
double end = 402;
double random = new Random().nextDouble();
double result = start + (random * (end - start));
System.out.println(result);
