JAVA - Read/write a .csv file 
writer = new PrintWriter("outFile.csv", "UTF-8");
writer.println("id,name");
writer.println("1,Bruce");
writer.println("2, John");
writer.close();

==>

Output Format

id,name
1,Bruce
2,John
