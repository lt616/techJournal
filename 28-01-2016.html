Bootstrap - Reference bootstrap
  1. <!DOCTYPE html>
  2. In head:
    <meta name="viewport" content="width=device-width, initial-scale=1">
  3. If wanna disable zooming function:
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
  4. Use bootstrap from Internet:
    <link rel="stylesheet" href="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css">
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script>
    <script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/js/bootstrap.min.js"></script> </li>

    Use bootstrap from local (only for this directory):
    <link rel="stylesheet" href="bootstrap/styles/bootstrap.min.css">
    <script src="bootstrap/scripts/bootstrap.min.js"></script>
    <script src="bootstrap/scripts/jquery-2.1.4.min.js"></script>

code: 
<nav class="navbar navbar-default">
	<div class="container-fuild"> // for expanding the whole screen
		<div class="navbar-header"></div>
		<div class="collapse navbar-collapse"></div>
		<div class="navbar-left"></div>
		<div class="navbar-right"></div>
	>
>
  1. Brand image:
    class="navbar-brand"
  2. Forms: 
     
<form class="navbar-form" role(prompt)="search">
	<div class="form-group">
	 ..submit button.. 
	</div>
>
  3. Button:
    <button type="button class="btn btn-default navbar-btn> button name </button>
  4. Text: class="navbar-text"
  5. DropdownToggle: 
    <div class="dropdown">
	<a href="# class="dropdown-toggle" data-toggle="dropdown" role="button" aria-haspopup="true" aria-expanded="false"> Dropdown </a>
	<ul class="dropdown">
			.. li .. 
	</ul>
</div>
  6. Collapse: 
    <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#bs-example-navbar-collapse-1" aria-expanded="false">
	<span class="sr-only"> Toggle navigation </span>
	<span class="icon-bar"></span>
</button>

    class="navbar-fixed-top/bottom"
