Set Amazon EC2 for ABE research
  1. Install Linuxbrew, to install glib and openssl
  2. Install GMP && PBC under '/usr/local' by package
  3. Install JAVA by package
  4. Install Tomcat7 , setup server.xml, upload server_store.jks
  5. Upload FileServer_SSL and start tomcat7, check with http://localhost:80 && http://localhost:443
